Shortcuts

    Enter: Create

    Delete or Backspace: Delete

    Down: Previous line

    Up: Next line

    Tab: Indent

    Shift + Tab: Unindent

    Shift + Up: Move line up

    Shift + Down: Move line down

    Command + Enter: Toggle line


    Defaults Shortcuts

    Command + Left: Beginning of line

    Command + Right: End of line

    Shift + Left: Select to left

    Shift + Right: Select characters to right

    Shift + Control + Left: Select all to left

    Shift + Control + Right: Select all to right

    Command + F: Find
